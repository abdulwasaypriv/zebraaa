Frequently Asked

Do I need to fill all sections? ▼
Not at all! Only your GitHub username is mandatory. All other sections — bio, social links, tech stack, stats — are completely optional. Customize your profile exactly how you want it.
Do I need to know Markdown? ▼
No coding knowledge needed at all. GitGloss is a complete no-code solution. Simply fill in the fields, toggle your preferences, and copy/download the generated README.
How do I use the generated README? ▼
1. Go to github.com/new and create a repo with the EXACT same name as your GitHub username. 2. Make it public and initialize with a README. 3. Paste your generated code into the README.md file. 4. Commit changes — your profile is live!
Are the stats cards real-time? ▼
Yes! The stats cards are powered by the GitHub API and update dynamically every time someone views your profile. They reflect your actual, current GitHub activity.
Is this tool free? ▼
Completely free, forever. No account required, no watermarks, no limitations. GitGloss is open source and will always remain free for the community.
Can I customize the stats card themes? ▼
Yes! Choose from 9+ themes including Tokyo Night, Radical, Dracula, Gruvbox, One Dark, Cobalt, Synthwave, and more in the GitHub Stats section of the builder.
